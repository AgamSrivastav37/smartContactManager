# Smart Contact Manager
Created a dynamic website to manage contacts, It can manage contacts and user details. Modules covered in the project are User Management and Contact Management, User Management covers Login, Logout, and Register. In Contact Management covers showing contacts, updating contacts, adding contacts, and searching contacts, on the profile page user can add contact details like contact name, nickname, email, and work experience. we can delete and edit the contacts.
## Techstack:Spring Boot, Spring MVC, MySQL, Sprig Security, JPA-Hibernate, html,css,javaScript.
